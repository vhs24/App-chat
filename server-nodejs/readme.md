## To Run Project `App Chat Realtime`
1. `Install Git`
2. `Clone Project`
```sh
git clone https://github.com/ButHetMuc/Gavroche-chat-app.git
```
3. `Open source code, dowload dependencies libraries`
```sh
npm install
```
4. `Run project Server`
```sh
npm start
```
